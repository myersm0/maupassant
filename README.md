# A Study: Maupassant in Translation
Source for [myersm0.github.io/maupassant-study](https://myersm0.github.io/maupassant-study), an experiment in corpus linguistics and literary translation.

## Local development
```bash
brew install zola   # macOS
zola serve
```

## Related repositories
- [Isosceles](https://github.com/myersm0/isosceles) — parallel corpus for literary translation
- [Montre](https://github.com/myersm0/montre) — corpus query engine
- [TEI-Littré](https://github.com/myersm0/tei-littre) — structured edition of Littré's dictionary
